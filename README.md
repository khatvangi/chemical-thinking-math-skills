# Chemical Thinking: Math Course Resources

Resources for building a chemistry-math course based on embodied cognition and computational thinking principles.

## Folder Structure

```
chem-math-course/
├── 01-cognitive-foundations/     # Theoretical foundation
│   ├── embodied-cognition/       # Lakoff & Núñez materials
│   └── p-prims/                  # Knowledge-in-pieces framework
├── 02-pedagogical-models/        # Model courses to learn from
│   ├── mit-computational-thinking/  # Julia + Pluto notebooks [CLONED]
│   ├── feynman-lectures/         # Pedagogical philosophy
│   ├── six-ideas/                # Big ideas organization
│   └── matter-interactions/      # Computational physics
├── 03-linear-algebra/            # Core math resources
│   ├── strang-18.06/            # MIT Linear Algebra [CLONED]
│   └── 3blue1brown/             # Visual intuition
├── 04-math-for-chemistry/        # Chemistry-specific math
├── 05-interactive-simulations/   # PhET and tools
│   ├── phet-info/
│   └── tools/
├── 06-chemical-data/             # Data sources reference
├── 07-visualization/             # Animation and plotting tools
│   └── PlutoUI.jl/              # [CLONED]
├── 08-julia-pluto/               # Technical setup guides
├── 09-education-research/        # Literature and papers
└── references/                   # Master links file
```

## Quick Start

1. **Read the cognitive foundations** in `01-cognitive-foundations/` to understand the theoretical approach
2. **Explore MIT notebooks** in `02-pedagogical-models/mit-computational-thinking/` for the technical model
3. **Watch 3Blue1Brown** videos linked in `03-linear-algebra/3blue1brown/`
4. **Set up Julia/Pluto** using guide in `08-julia-pluto/`
