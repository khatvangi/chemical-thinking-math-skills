# Chemical Data Sources

## NIST Chemistry WebBook (Primary)
- **Main Portal:** https://webbook.nist.gov/chemistry/
- **Search:** https://webbook.nist.gov/
- **Data Types:**
  - Thermodynamic data (Î”H, Cp, phase transitions)
  - Spectroscopy (IR, MS, UV-Vis)
  - Search by name, formula, or CAS number

## Molecular Structure Databases

### PubChem
- **URL:** https://pubchem.ncbi.nlm.nih.gov/
- **Use:** Molecular structures, properties, safety data
- **Features:** Free, comprehensive, API available

### ChemSpider
- **URL:** http://www.chemspider.com/
- **Use:** Structure search, property data
- **Features:** Links to original sources

## Spectroscopy Databases

### SDBS (Spectral Database for Organic Compounds)
- **URL:** https://sdbs.db.aist.go.jp/
- **Data:** IR, NMR, MS, ESR, Raman spectra
- **Note:** Free registration required

## Usage in Course
- Use NIST for thermodynamic calculations
- PubChem for molecular structure data
- Real data for problem sets and examples
