# Mathematical Visualization Tools

## Primary Tools

### Desmos
- **URL:** https://www.desmos.com/calculator
- **Use:** Function manipulation, graphing, sliders
- **Features:** Free, browser-based, shareable links

### GeoGebra
- **URL:** https://www.geogebra.org/
- **Use:** 3D geometry, vectors, transformations
- **Features:** 3D graphing, CAS, geometry tools

### MolView
- **URL:** https://molview.org/
- **Use:** 3D molecular visualization
- **Features:** SMILES input, structure search, interactive 3D

## For Course Development
- Create interactive demonstrations with Desmos
- Use GeoGebra for vector/matrix visualizations
- MolView for molecular geometry connections to math
